# DATA — Shared Datasets & Captures

## PURPOSE
Stores data used across the project.

This folder allows reproducibility without hardware.

---

## CONTENTS
- Sample VOC datasets
- Captured sensor windows
- Test JSON payloads

---

## RULES
- Keep files small
- Do NOT upload massive raw datasets
- Only commit demo-sized samples

---

## USE CASES
- Backend testing without hardware
- Frontend demo fallback
- ML experimentation

---

## SUCCESS CONDITION
Project can run demo using stored sample data if hardware fails.
